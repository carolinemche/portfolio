PROCESSING/P5.JS

Voogle is a platform created on P5.js and hosted on Netlify where users can use keywords to search up specific Vines. Since the deletion of the original platform, there has been no way to conveniently find a Vine. This platform allows for users to easily access a vine based on keywords.

Generative Sea is a landscape created on Processing that is an animation that depicts the sea changing overtime due to plastic waste. When the user clicks on the landscape, plastic waste such as straws and cans are shown deteriorating the sea.

GetTech'd is an online platform for girls and young women interested in technology. It provides them with resources and opportunities to reach out to professionals in the field. It was created as a final project the Girls Who Code Summer Immersion Program at AIG. It utilizes HTML, CSS, and Javascript.
